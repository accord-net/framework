The images available in this folder are part of the "101 Object Categories"
database from the California Institute of Technology. Samples from this
database can be found on:

 - http://pascallin.ecs.soton.ac.uk/challenges/VOC/thumbs/101/index.html

Publications using this database are:

 - L. Fei-Fei, R. Fergus, and P. Perona. Learning generative visual models from
   few training examples: an incremental Bayesian approach tested on 101 object
   categories. In Proceedings of the Workshop on Generative-Model Based Vision.
   Washington, DC, June 2004.
